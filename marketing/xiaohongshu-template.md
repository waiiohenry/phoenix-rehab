# 📱 小红书内容模板

## 爆款笔记结构

```
【标题】痛点 + 解决方案 + 数字
- 字数: 20-30字
- 开头3行必须抓住眼球

【正文】300-500字
- 第一段: 共鸣 (你的问题我懂)
- 中间: 干货价值
- 结尾: CTA

【标签】5-8个
- 行业标签: #康复理疗 #物理治疗 #肩颈疼痛
- 流量标签: #打工人 #久坐族 #温哥华
```

## 标题模板

| 类型 | 例子 |
|------|------|
| 痛点型 | "肩颈酸痛到睡不着？3个动作救你" |
| 数字型 | "7天改善腰疼｜打工人必看" |
| 疑问型 | "你的腰疼可能是这个原因？" |
| 对比型 | "中西结合康复｜效果翻倍" |

## 内容模板

### 模板1：康复Tips
```
标题: 【痛点】救星｜【数字】个动作

大家好，我是XX医生/治疗师

今天分享【主题】

动作1: xxx
- 要点: xxx
- 常见错误: xxx

动作2: xxx
...

坚持【时间】，你会感受到变化！

有问题评论区问我！

#康复 #打工人 #健康
```

### 模板2：患者故事
```
标题: 从【之前状态】到【之后状态】

今天分享一个患者的康复故事

【背景】
- 年龄/职业/问题
- 困扰了多久

【康复过程】
- 评估发现
- 治疗方案
- 关键转折

【结果】
- 数据对比
- 患者反馈

【你的感悟】
- 什么帮助最大
- 建议给类似问题的人

#患者故事 #康复 #真实案例
```
