## 快速参考

### 标题公式

**效果+人群型**
```
公式: [强效果] + [特定人群] + [承诺]
示例: "去黄皮太狠了！黑黄皮的逆袭，素颜好皮"
```

**反差悬念型**
```
公式: [否定常识] + [真实结果] + [表情]
示例: "居然有人说XXX！我真的会谢..."
```

**权威干货型**
```
公式: [数字] + [解决方案] + [权威身份]
示例: "3点定位法，如何快速解决法令纹？医生亲授"
```

---

### 违禁词速查

| 违禁 | 安全替代 |
|------|----------|
| 治疗/治愈 | 调理/养护 |
| 根治 | 改善/缓解 |
| 消炎 | 舒缓/去红 |
| 排毒 | 轻体/净化 |
| 第一/最好 | 天花板/宝藏 |

---

### 封面最佳实践

- **尺寸**: 3:4 竖屏 (1242 x 1660像素)
- **对比法**: 前后对比最有冲击力
- **大字报**: 关键信息放大
- **真实感**: 2025年"活人感"更受欢迎

---

### 核心算法指标

CES评分权重 (从高到低):
1. 转发 (4分)
2. 评论 (4分)
3. 收藏 (1分)
4. 完播率 (1分)
5. 点赞 (1分)
